# gcp-data-pipelines

A learning repository following the **Data Engineering Zoomcamp 2026** by DataTalksClub.  
Covers hands-on modules across Python, Kestra, Terraform, Docker, dbt, Spark, and GCP.

> **Note:** This repo is for learning and notes. My portfolio project will be built after completing all modules.

## Modules Covered
- Module 1: Containerization & Infrastructure (Docker, Terraform)
- Module 2: Workflow Orchestration (Kestra)
- Module 3: Data Warehousing (BigQuery)
- Module 4: Analytics Engineering (dbt)
- Module 5: Batch Processing (Spark)
